# docs/ 目录说明

此目录存放文档资源。

## demo.gif

演示 GIF 尚未生成。录屏后请将 GIF 文件命名为 `demo.gif` 放到此目录，
然后取消注释 README.md 第 15 行的 GIF 嵌入代码：

```markdown
![演示 GIF](docs/demo.gif)
```

## 录制建议

1. 参考 `DEMO_SCRIPT.md` 的场景三（零摩擦捕获）和场景四（知识库推荐）录制
2. 控制在 30~60 秒，覆盖核心亮点
3. 推荐工具：
   - macOS：[Kap](https://getkap.co/) 或 [gifox](https://gifox.io/)
   - Linux：[peek](https://github.com/phw/peek)
   - Windows：[ScreenToGif](https://www.screentogif.com/)
4. 分辨率建议 1280×720，帧率 10fps（平衡文件大小与清晰度）
5. GIF 大小控制在 5MB 以内（GitHub README 加载友好）
